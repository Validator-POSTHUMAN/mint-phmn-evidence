Русскоязычная версия для публикации в GitHub и приёма контр-доказательств через pull requests.

## Как использовать этот документ

- Этот документ объединяет техническую лестницу доказательств и описание разговора с Олимом.
- Мы открыты к контр-доказательствам, исправлениям, уточнениям и альтернативным рациональным интерпретациям.
- Если вы считаете какой-то пункт неверным или слабым, присылайте pull request с конкретным исправлением и ссылками на проверяемые источники.
- Не публикуйте seed phrase, private keys, mnemonic, пароли, токены доступа или приватные логи без редактирования чувствительных данных.
- Контр-доказательства должны быть проверяемыми: timestamps, tx hashes, IP/SSH fingerprints, signed statements, screenshots с контекстом, экспорт логов или иные воспроизводимые артефакты.
- Этот документ не является юридическим заключением и не доказывает личность человека сам по себе. Он фиксирует факты, ограничения и текущую интерпретацию.
